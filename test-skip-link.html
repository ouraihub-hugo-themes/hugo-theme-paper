Skip to Content Link Test

测试说明

这个页面用于测试 "Skip to Content" 链接的功能。

  1. 按 Tab 键：Skip link 应该是第一个获得焦点的元素
  2. 获得焦点时：Skip link 应该从顶部滑入并显示
  3. 点击 Skip link：页面应该跳转到主内容区域（这个标题下方）
  4. 失去焦点后：Skip link 应该再次隐藏

为什么需要 Skip Link？

Skip to content 链接是一个重要的可访问性功能，它允许键盘用户和屏幕阅读器用户快速跳过导航菜单，直接到达页面的主要内容。

这对于以下用户特别有用：

  • 使用键盘导航的用户
  • 使用屏幕阅读器的视障用户
  • 运动障碍用户

实现细节

我们的实现完全参考了 AstroPaper 的设计：

  • 默认隐藏（position: absolute; top: -100%）
  • 获得焦点时显示（focus:top-4）
  • 使用 backdrop-filter 实现毛玻璃效果
  • 平滑的过渡动画
  • 清晰的焦点指示器

测试结果

如果你能看到这个页面，并且按 Tab 键后能看到 "Skip to content" 链接出现在页面顶部，那么实现是成功的！
