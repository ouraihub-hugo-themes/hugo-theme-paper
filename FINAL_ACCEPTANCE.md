## 附录: 快速参考

### 快速开始
```bash
cd hugo-paper
pnpm install
pnpm dev        # 开发模式
pnpm test       # 运行测试
pnpm build      # 生产构建
```

### 关键文件位置
- 配置: `hugo.toml`, `params.toml`
- 模板: `layouts/` 目录
- 样式: `assets/css/`, 配置在 `tailwind.config.js`
- 脚本: `assets/ts/main.ts`
- 测试: `assets/ts/*.test.ts`
- 文档: `*.md` 文件

### 测试命令
```bash
pnpm test              # Watch 模式
pnpm test:run         # 单次运行
pnpm test:coverage    # 覆盖率报告
pnpm test:ui          # UI 界面
```
